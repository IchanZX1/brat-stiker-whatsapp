# Samsung emoji assets

Put Samsung Android 14 / One UI 6 emoji assets here if you have licensed files.

Supported filenames:

- `1f602.svg`
- `1f602.png`
- `emoji_u1f602.svg`
- `emoji_u1f602.png`

For emoji sequences, use hyphen style (`1f468-200d-1f4bb.svg`) or Noto style
(`emoji_u1f468_200d_1f4bb.svg`).

When a matching local asset exists, the API uses it before the default Noto
Android-style emoji fallback.
